# list2json

Convert multilevel list to JSON.

## Install

```
git clone https://github.com/huangshaolin/list2json.git
cd list2json
pip3 install --user -r requirements.txt
```

## Usage

```
cd list2json
python3 list2json.py [--debug] <list_file_path> [<json_path>] [--indent=<indent_str>]
```
